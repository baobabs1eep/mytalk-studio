## 表单填写建议

### 队伍名称

推荐：**MyTalk Studio**

### 项目名称

**MyTalk Studio：从 TED Talk 到 My Talk 的表达成长平台**

### 一句话介绍

从 TED 的一个观点出发，帮助年轻人留下真实感触、形成个人表达，并在彼此看见中持续成长。

## Demo 链接的处理

本地 HTML 不能保证评委直接打开。优先顺序：

1. 上传到 GitHub Pages 或 Vercel，提交公开网址。
2. 上传 HTML 至飞书云文档或网盘，确保“持链接可查看/下载”。
3. 无法上传时，至少提交 PDF + 演示录屏，并在附件说明中注明“可交互原型见随附 HTML”。

## 录屏讲解提纲（约 2 分 30 秒）

1. **0:00 - 0:20：问题**。许多学生看完 TED 有感触，却不知道如何说，也缺少被看见的动力。
2. **0:20 - 0:45：足迹**。先记录一句没有发出去的话；它无需完整，之后可以编辑、打标签、回看。
3. **0:45 - 1:25：表达工坊**。以 TED 感受进入，选择足迹，经过 AI 追问，形成自己的表达草稿。
4. **1:25 - 1:55：思想图书馆**。将作品制作成封面并分享；他人可以认同、追问和收藏，形成真实反馈。
5. **1:55 - 2:20：成长主页**。足迹、成稿、主题和表达日历让持续积累可见。
6. **2:20 - 2:30：结尾**。TED 是思想入口，MyTalk Studio 帮助学生把启发变成自己的声音。
