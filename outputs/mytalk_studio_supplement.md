# MyTalk Studio 项目补充说明

## 1. 项目名称与主张

**MyTalk Studio：从 TED Talk 到 My Talk 的表达成长平台**

从 TED 的一个观点出发，帮助年轻人留下真实感触、形成个人表达，并在彼此看见中持续成长。

## 2. 我们看见的问题

看完一场 TED，心里很亮，却说不出为什么；刷到一句话，明明想回应，最后还是退出了；电影散场后有很多感受，却只发了一个“好看”。这些时刻并不缺少想法，只缺少一个能慢慢把话说完的地方。

学生的表达困难通常来自三件事：

1. 感触停留在片段，缺少整理与展开的入口。
2. 害怕说不清、被误解，公共表达的心理门槛很高。
3. 一次展示结束后，个人经历与表达进步没有被持续记住。

## 3. 命题回应

TED 为青年提供了高质量的全球思想输入。MyTalk Studio 以 TED 为重要入口，同时允许用户把书、电影、校园、家庭与网络讨论中的感触带进来。这样，学生不会只停留在理解一场演讲，而会将外部观点与自身经验连接，形成愿意说出的 My Talk。

## 4. 产品闭环

### 4.1 足迹：留下还没说出口的话

用户记录一句感受、一个情绪、一个疑问或一条没有发出的回应。纸条可编辑、打标签、回看，也能收藏图书馆中他人的表达。足迹是后续表达最真实的素材库。

### 4.2 表达工坊：陪用户把一个念头说清楚

用户可选择四种入口：

- 我只有一个瞬间，还说不清
- 我已经有一个想法
- 把 TED 感受变成自己的话
- 回应一个让我想反驳的观点

系统通过多轮问题帮助用户追问“我为什么会这样想”，并允许选取 1 至 3 条足迹带入。用户可选择生成一句话、一段话、1 分钟发言或 3 至 5 分钟 My Talk；草稿的标题、核心观点和正文均可继续编辑。

### 4.3 思想图书馆：让表达被温和地看见

用户将作品制作成封面后分享进入图书馆。阅读者可以表达认同、留下温和追问、收藏一条足迹，再把被触动的想法带回自己的表达工坊。这里像一座可阅读的观点图书馆，也让表达具有真实的人际反馈。

### 4.4 成长主页：让积累变得可见

个人主页记录足迹、成稿、分享、常出现主题和表达日历。它让用户知道自己关心什么、哪些观点重复出现、表达正在怎样变化；持续学习由此从外部任务变成可感知的个人积累。

## 5. 用户体验示例

一位大学生看完一场关于 AI 与创造力的 TED 演讲，想到课堂上“只奖励标准答案”的经历，却不知道如何展开。她在足迹中写下一句反驳，随后在表达工坊选择“把 TED 感受变成自己的话”，带入两条相关足迹。AI 依次追问她看到的具体场景、真正介意的部分、希望他人重新思考的问题。她最终形成一篇题为《我为什么仍然需要提出问题》的 My Talk，并分享至思想图书馆。一次表达完成后，它也成为她成长主页里“AI”“教育”主题的一部分。

## 6. 差异化

| 常见产品 | 核心动作 | MyTalk Studio 的做法 |
| --- | --- | --- |
| TED 内容推荐 | 看更多内容 | 从 TED 观点进入个人表达 |
| TED 摘要工具 | 更快看懂内容 | 追问用户与内容的关系 |
| 口语陪练 | 训练发音、语法 | 先形成真正想说的内容，再练表达 |
| AI 写作工具 | 直接产出标准文本 | 引用足迹、保留用户编辑权与表达过程 |
| 单次演讲课程 | 完成一次任务 | 用足迹、图书馆与成长主页形成持续循环 |

## 7. 对新航道 TED 的价值

- 让 TED 内容从观看资源进入学生的日常表达训练。
- 将英语学习、批判性思维、公共表达与自我认知放在同一条成长路径中。
- 通过作品分享与主题沉淀，提高学生持续使用和持续学习的意愿。
- 形成可展示的青年表达作品集，为课程、社团、演讲活动提供延展空间。

## 8. Demo 说明

本项目已完成可交互 Web 原型，展示完整的“足迹 - 工坊 - 图书馆 - 成长”闭环。原型为前端演示版本：内容生成与反馈使用可控的本地模拟逻辑，以便聚焦验证产品流程、交互和价值主张；正式版本可接入大语言模型、TED 字幕/文本解析、语音识别与语音反馈能力。
